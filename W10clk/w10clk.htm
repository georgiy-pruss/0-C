Actions

f1     show help
?      show help
h      show help (when display is clear)

Nh     show number (integer part) in hexadcimal system

d      get current date as y/m/d wd (wd - weekday, Mon to Sun)
Nd     convert CED as plain number to y/m/d wd diff, e.g. 730122 --> 2000/1/1 Sat
±Nd    add or substract days from the current day, giving date in the future or past
MDd    convert m/d to n wd [diff], meaning the current year
YMDd   convert y/m/d to n wd [diff], e.g. 2016/11/13 --> 736283 Sun

u      current unix time (for local time)
Nu     unix time to y/m/d_h:m:s wd (with the current tz!), in range 1970/1/1 to 2106/2/7

t      seconds (of day) now, 0 to 86400 (the even 86400 can be the very rare case of leap second)

y      current year day number (1..366)

z      current time zone as +HHMM or -HHMM, e.g. +0200

Nc     convert °Celsius to °Farenheit
Nf     convert °Farenheit to °Celsius

s      start stopwatch (actually just remember the moment of start)
e      elapsed since start, also can show break time and work time
b      break (don't count the following period, start the break)
g      go on (continue working or whatever, end of break)

ctrl+C copy contents of display to clipboard
ctrl+V paste clipboard into display

ctrl+T toggle showing time in window caption bar aka title (option 'intitle')
tab    toggle showing seconds hand, same as right button action (option 'seconds')

c      show color dialog (just to have it easily accessible), colors are #BBGGRR!!!

=      calculate expression (see below how to enter them)
enter  calculate expression

Input

0-9.   enter number, e.g. 42 or 3.1415
+-     can be used for numbers with sign, e.g. +10 (sometimes it's not the same as 10)
:/_    enter time, dates, e.g. 3/11 or 2:00 or 2016/11/10_23:59:59
ABCDEF hexadecimal digits (uppercase!) for hex. numbers in expression, put # in front
#      hexadecimal number indicator: #FF #DEADBEEF #100000000

       binary and unary operations in math. expression (priority: ^ */%o +-)
+      +x  the same as x           x+y  sum
-      -x  negate                  x-y  subtract
*      *x  square, x*x             x*y  multiply
/      /x  square root of x        x/y  divide
%      %x  natural logarithm of x  x%y  modulo
^      ^x  exponent                x^y  power (left to right order, sorry)
o      ox  multiply by pi          xoy  circle function (hello APL ;-))

           1ox sin    -1ox asin    10ox sind    -10ox asind
           2ox cos    -2ox acos    20ox cosd    -20ox acosd
           3ox tan    -3ox atan    30ox tand    -30ox atand

()     just usual grouping order of evaluation with ()

bksp   correct input (delete the character at the end of display)
del    correct input (delete the character at the beginning)

spc    clear display
esc    clear display


CED    Common Era Day number -- number of a day since January 1st, 1 AD, i.e.
       that was day number 1, then January 2nd was day 2 and so on, using Julian
       calendar (year divisable by 4 are all leap), up till 1582/10/4 (CED 577737),
       when the counting scheme switches to Gregorian calendar and day 1582/10/15
       (CED 577738) follows. Such enumeration was chosen to eliminate negative
       numbers and to have somewhat natural scheme.
       CED 730122 = Julian Day Number 2451545 (the difference is 1721423)

Screensaver

You can also set up it as a screensaver. Copy the executable to some other file with extension .scr, e.g. w10clkss.scr. Copy or create config file for it, it must be named then w10clkss.ini. Probably it should be not resizable, no time in title, better black, etc. For example:

[window]
xywh=10,10,1900,1060
bg=0,0,0
ticks=12,40,40,40
shand=7,40,40,40
mhand=11,44,42,44
hhand=22,44,42,44
intitle=yes
resizable=no
ontop=yes

Then right-click on the scr file and choose Install. It will also start at once. Press 'q' to stop it. It has no usual config dialog, like other screensavers, all the setup is done in ini file. Anyway, it'll serve as a screensaver. To exit it, press 'q', no other way works. Moreover, it functions with all its features — date/time, stopwatch, calculations, etc.
